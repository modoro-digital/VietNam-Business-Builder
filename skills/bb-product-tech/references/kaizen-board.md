### PROMPT 11: Kaizen Board (Bảng Cải tiến Liên tục)

#### Mô tả
Bảng quản lý ý tưởng và dự án cải tiến liên tục theo triết lý Kaizen — từ đề xuất → đánh giá → ưu tiên → triển khai → đo lường. Tạo văn hóa cải tiến ở mọi cấp trong DN.

#### Thông tin cần thu thập (hỏi người dùng trước khi tạo)
1. DN đã có văn hóa cải tiến chưa? (formal / informal / chưa có)
2. Ai có quyền đề xuất cải tiến? (tất cả NV / chỉ quản lý)
3. Có reward/recognition cho cải tiến được áp dụng không?
4. Tần suất review ý tưởng cải tiến? (hàng tuần / hàng tháng)
5. Lĩnh vực nào cần cải tiến nhất? (quy trình / sản phẩm / chi phí / trải nghiệm KH)

#### Template gợi ý
Cấu trúc MAN:
- **Phần 1** — Triết lý Kaizen trong DN: Tại sao, nguyên tắc, văn hóa
- **Phần 2** — Quy trình đề xuất cải tiến: Flowchart đề xuất → screening → đánh giá → ưu tiên
- **Phần 3** — Kaizen Board Layout: Backlog → In Review → Approved → In Progress → Completed → Measured
- **Phần 4** — Tiêu chí đánh giá: Impact × Effort matrix (Quick Wins / Major Projects / Fill-ins / Hard Slogs)
- **Phần 5** — Tracking & Metrics: Số đề xuất, % implemented, cost saved, time saved
- **Phần 6** — Recognition Program: Cơ chế khen thưởng cải tiến
- **Phụ lục**: Improvement Proposal Form (link PROMPT 12), Kaizen event template

Confirm cấu trúc trước khi tạo.

#### Prompt tạo file
```
Tạo Kaizen Board (Bảng Cải tiến Liên tục).

CONTEXT:
- DN: [Tên] — Văn hóa cải tiến: [formal / informal / chưa có]
- Ai đề xuất: [tất cả NV / quản lý]
- Review: [tần suất]
- Reward: [Có/Không] — [chi tiết]
- Focus areas: [quy trình / SP / chi phí / CX]

FORMAT:
- Kaizen board layout: Mermaid / Kanban — 6 cột swim lanes
- Submission flowchart: Mermaid — Ý tưởng → Form → Review → Prioritize → Implement → Measure
- Impact-Effort matrix: 2×2 grid — Quick Wins | Major Projects | Fill-ins | Thank but No
- Scoring template: Criteria | Weight | Score (1-5) → Priority
  - Impact on customer, Impact on cost, Effort, Speed to implement, Strategic alignment
- Metrics dashboard: KPIs — # submitted | # approved | # completed | Avg cycle time | Total savings
- Recognition program: Tier | Criteria | Reward | Examples
- Monthly Kaizen report template: Summary + top 5 + metrics + next month focus

TONE: Motivational + structured — khuyến khích tham gia, nhưng có quy trình rõ.
LENGTH: 5-8 trang A4.
```
