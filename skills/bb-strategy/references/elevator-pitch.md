# PROMPT 04: Elevator Pitch (Bài giới thiệu ngắn)

#### Mô tả
Bộ elevator pitch cho nhiều ngữ cảnh: 30 giây (networking), 60 giây (nhà đầu tư), 90 giây (đối tác). Mỗi version được customize cho đối tượng khác nhau. CEO và team sales phải thuộc lòng.

#### Thông tin cần thu thập (hỏi người dùng trước khi tạo)
1. Một câu mô tả DN dễ hiểu nhất? ("Chúng tôi giúp [ai] [làm gì] bằng [cách gì]")
2. Điểm khác biệt lớn nhất so với đối thủ? (1-2 câu)
3. Proof point mạnh nhất? (số liệu / khách hàng / giải thưởng)
4. CTA mong muốn? (gặp lại / demo / gửi proposal / follow social)
5. Cần version nào? (Nhà đầu tư / Khách hàng / Đối tác / Tuyển dụng / Media / Tất cả)
6. Ngôn ngữ: Tiếng Việt / Tiếng Anh / cả hai?

#### Template gợi ý
Cấu trúc FRM:
- **30-Second Pitch** (Networking/Elevator) — Hook → Problem → Solution → CTA
- **60-Second Pitch** (Investor/Partner) — Hook → Problem → Solution → Traction → Ask
- **90-Second Pitch** (Deep Intro) — Hook → Problem → Solution → Why us → Traction → Vision → CTA
- **Customize per audience**: Nhà đầu tư (focus: market size, traction, ROI) / Khách hàng (focus: pain point, benefit, proof) / Tuyển dụng (focus: mission, culture, growth) / Media (focus: story, impact, uniqueness)
- **Tips & Practice Guide** — Cách luyện tập, ngôn ngữ cơ thể, common mistakes

Confirm cấu trúc trước khi tạo.

#### Prompt tạo file
```
Tạo bộ Elevator Pitch cho doanh nghiệp.

CONTEXT:
- DN: [Tên] — Ngành: [ngành]
- 1-liner: "[Chúng tôi giúp [ai] [làm gì] bằng [cách gì]]"
- USP: [điểm khác biệt]
- Proof points: [số liệu/khách hàng/giải thưởng]
- Audiences: [Investor / Customer / Partner / Recruit / Media]
- Ngôn ngữ: [VI / EN / cả hai]

FORMAT:
- 3 versions: 30s (~75 từ) / 60s (~150 từ) / 90s (~225 từ)
- Customize per audience: 5 audiences × 3 lengths = 15 variations (hoặc chọn combo)
- Mỗi pitch: Script đầy đủ + ghi chú (nhấn mạnh từ nào, pause ở đâu, eye contact khi nào)
- Đánh giá: ☐ Clear? ☐ Memorable? ☐ Differentiating? ☐ Has CTA? ☐ Under time?
- Practice guide: Cách luyện trước gương, record & review, A/B test

TONE: Tự tin, conversational, không đọc thuộc lòng. Giọng kể chuyện.
LENGTH: 3-5 trang A4.
```
