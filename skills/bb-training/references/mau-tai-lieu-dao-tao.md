### PROMPT 06: Training Material Template (Mẫu Tài liệu Đào tạo)

#### Mô tả
Template chuẩn hóa cho tất cả tài liệu đào tạo trong DN — bao gồm cấu trúc bài giảng, slide deck, handout, và facilitator guide. Đảm bảo tính nhất quán, chuyên nghiệp, và dễ bảo trì/cập nhật. Áp dụng nguyên lý thiết kế học tập (instructional design) theo ADDIE model.

#### Thông tin cần thu thập (hỏi người dùng trước khi tạo)
1. Format tài liệu đào tạo chủ yếu? (slide / document / video script / e-learning)
2. Có branding guidelines không? (logo, màu sắc, font)
3. Ai là đối tượng chính sử dụng template? (trainer nội bộ / external trainer / self-study)
4. Cần bao gồm những phần nào? (objectives, content, activities, assessment, handout)
5. Thời lượng trung bình một buổi đào tạo? (1h / half-day / full-day)
6. Có cần phần đánh giá/quiz trong tài liệu không?

#### Template gợi ý
Cấu trúc FRM:
- **Phần 1** — Cover Page Template: Tên khóa học, trainer, date, version, branding
- **Phần 2** — Course Overview: Learning objectives (SMART), target audience, prerequisites, duration
- **Phần 3** — Facilitator Guide Template: Timeline, instructions, talking points, activity facilitation
- **Phần 4** — Content Slides Template: Cấu trúc mỗi module — hook, content, practice, review
- **Phần 5** — Participant Handout Template: Key takeaways, exercises, notes space, resources
- **Phần 6** — Assessment Template: Pre-test, post-test, quiz, practical exercise
- **Phần 7** — Supplementary Materials: Case studies, role-play scenarios, worksheets
- **Phụ lục**: Branding guidelines for training, File naming convention

Confirm cấu trúc trước khi tạo.

#### Prompt tạo file
```
Tạo Training Material Template (Mẫu Tài liệu Đào tạo).

CONTEXT:
- DN: [Tên] — Format chính: [slide / document / e-learning]
- Branding: [Có/Không] — Logo: [Có] — Brand colors: [mã màu]
- Người dùng template: [trainer nội bộ / external / self-study]
- Thời lượng trung bình: [X giờ]
- Assessment: [Có/Không]

FORMAT:
- Cover page template: [Tên khóa] | [Trainer] | [Date] | [Version] | [Logo] — Layout mockup
- Course overview template:
  - Learning objectives: "Sau khóa học, học viên có thể..." — dùng Bloom's Taxonomy verbs
  - Audience profile: [___] | Prerequisites: [___] | Duration: [___]
  - Agenda: Module | Topic | Duration | Method | Materials
- Facilitator guide template per module:
  - Time | Slide # | Trainer says/does | Learner does | Materials | Notes
  - Transition cues, activity instructions, debrief questions
- Content template: Mỗi module — Hook (câu hỏi/story) → Teach (content) → Practice (activity) → Review (summary)
- Slide design rules:
  - Font size min | Max bullets per slide | Image guidelines | Branding placement
  - Template slides: Title, Content, Activity, Quote, Case study, Q&A
- Handout template: Module | Key points | Notes space | Action items | Resources/Links
- Assessment templates:
  - Pre-test: 5-10 câu đo baseline
  - Post-test: 5-10 câu đo learning gain
  - Practical exercise: Scenario | Task | Expected output | Evaluation criteria
- File naming convention: [Dept]-[Topic]-[Type]-[Version]-[Date] — Ví dụ: HR-Onboarding-FacGuide-v1.0-2024Q1

TONE: Professional, dễ customize, instructional design–driven.
LENGTH: 4-6 trang A4.
```
