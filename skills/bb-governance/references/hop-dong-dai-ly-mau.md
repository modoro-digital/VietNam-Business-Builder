# PROMPT 15: Hợp đồng đại lý mẫu (Agency/Distribution Agreement Template)

#### Mô tả
Template hợp đồng đại lý/phân phối — quy định commission, territory, exclusivity, quota, termination. Dùng cho mạng lưới đại lý/nhà phân phối.

#### Thông tin cần thu thập (hỏi người dùng trước khi tạo)
1. Mô hình phân phối? (Đại lý bán hàng / Nhà phân phối mua đi bán lại / Affiliate)
2. Exclusive hay Non-exclusive?
3. Territory: theo tỉnh/vùng/toàn quốc?
4. Commission structure: % / fixed / tiered?
5. Có quota (chỉ tiêu) bắt buộc không?
6. Sản phẩm/dịch vụ phân phối là gì?

#### Template gợi ý
Cấu trúc FRM:
- **Điều 1** — Các bên, Bổ nhiệm đại lý
- **Điều 2** — Sản phẩm/Dịch vụ, Territory, Exclusivity
- **Điều 3** — Quyền & Nghĩa vụ: Bên giao (Principal) + Bên đại lý (Agent)
- **Điều 4** — Giá, Chiết khấu, Commission
- **Điều 5** — Quota & Performance: Chỉ tiêu tối thiểu, review kỳ
- **Điều 6** — Marketing & Thương hiệu: Co-branding rules
- **Điều 7** — Đặt hàng, Giao hàng, Kho, Trả hàng
- **Điều 8** — Thanh toán: Kỳ hạn, phương thức, chiết khấu thanh toán
- **Điều 9** — Bảo mật & Non-compete
- **Điều 10** — Thời hạn, Gia hạn, Chấm dứt
- **Phụ lục**: Bảng giá, Bảng commission, Territory map, KPI form

Confirm cấu trúc trước khi tạo.

#### Prompt tạo file
```
Tạo Template hợp đồng đại lý (Agency/Distribution Agreement).

CONTEXT:
- DN: [Tên] — Sản phẩm: [mô tả]
- Mô hình: [Đại lý / Nhà PP / Affiliate]
- Exclusive: [Có/Không]
- Territory: [scope]
- Commission: [structure chi tiết]
- Quota: [Có/Không] — Mức: [chi tiết]

FORMAT:
- Template form: Trường điền = [___]
- Bảng commission: Level | Doanh số từ-đến | % Commission
- Territory map placeholder
- KPI dashboard: Metric | Target | Actual | Achievement %
- Quy trình đặt hàng: Flowchart

TONE: Thương mại, cân bằng, rõ ràng.
LENGTH: 8-12 trang A4 + phụ lục.

CROSS-REFERENCE: Tài liệu này là template pháp lý chuẩn cho hợp đồng đại lý. Phiên bản tập trung vào quan hệ đối tác thương mại xem bb-sales/Partner Agreement Template.
```
