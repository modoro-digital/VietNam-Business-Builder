## [0.3.0] — 2026-04-30

### Added
- 🎉 **Initial public release** trên GitHub.
- Bộ **13 skills** với **191 prompt files** theo kiến trúc 5 tầng:
  - **Tầng Meta** — `bb-orchestrator` (điều phối tổng thể)
  - **Tầng 1 (Nền móng)** — `bb-governance`, `bb-strategy`
  - **Tầng 2 (Bộ máy)** — `bb-finance`, `bb-people`, `bb-operations`
  - **Tầng 3 (Động cơ)** — `bb-sales`, `bb-marketing`, `bb-customer`
  - **Tầng 4 (Tối ưu)** — `bb-product-tech`, `bb-training`, `bb-reporting`
  - **Tầng 5 (Nhân rộng)** — `bb-growth`
- File `BLUEPRINT.md` mô tả toàn bộ kiến trúc tổng thể (v2.0).
- Plugin manifest `.claude-plugin/plugin.json` tương thích Claude Code/Cowork.
