# Changelog

Tất cả thay đổi đáng chú ý của plugin **Business Builder** sẽ được ghi tại đây.

Format theo [Keep a Changelog](https://keepachangelog.com/en/1.1.0/), tuân thủ [Semantic Versioning](https://semver.org/lang/vi/).

## [0.3.0] — 2026-04-30

### Added
- 🎉 **Initial public release** trên GitHub.
- Bộ **13 skills** với **204 prompt files** theo kiến trúc 5 tầng:
  - **Tầng Meta** — `bp-orchestrator` (điều phối tổng thể)
  - **Tầng 1 (Nền móng)** — `bp-governance`, `bp-strategy`
  - **Tầng 2 (Bộ máy)** — `bp-finance`, `bp-people`, `bp-operations`
  - **Tầng 3 (Động cơ)** — `bp-sales`, `bp-marketing`, `bp-customer`
  - **Tầng 4 (Tối ưu)** — `bp-product-tech`, `bp-training`, `bp-reporting`
  - **Tầng 5 (Nhân rộng)** — `bp-growth`
- File `BLUEPRINT.md` mô tả toàn bộ kiến trúc tổng thể (v2.0).
- Plugin manifest `.claude-plugin/plugin.json` tương thích Claude Code/Cowork.
